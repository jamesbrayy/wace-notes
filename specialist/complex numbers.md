# **polynomials:**
### *factor theorem:*
  - if $$f\left(a\right)=0$$, then $$\left(x-a\right)$$ is a factor $$f\left(x\right)$$
### *remainder theorem:*
  - if $$f\left(x\right)$$ is divided by $$(x-a)$$, the remainder will be $$f\left(a\right)$$
